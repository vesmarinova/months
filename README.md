# Months & Seasons Learning Game 📅

An interactive, mobile-friendly educational game teaching children (ages 5-10) the sequence of months and their associated seasons. Bilingual support for English and Bulgarian.

**Live Demo**: https://vesmarinova.github.io/months/

## Tech Stack

- **React 19** + **TypeScript** (strict mode)
- **Vite 8** for build tooling
- **Tailwind CSS 4** for styling
- **@dnd-kit** for accessible drag-and-drop
- **GitHub Pages** for deployment

## Getting Started

```bash
npm install
npm run dev      # Start dev server
npm run build    # Production build
npm run preview  # Preview production build
npm run deploy   # Deploy to GitHub Pages
```

## Game Modes

- **Month Sequence** (✅ Implemented): Drag-and-drop months into the correct order (January → December)
- **Season Sorting** (🔜 Coming Soon): Match months to their seasons
- **Quick Quiz** (🔜 Coming Soon): Multiple choice knowledge test

## Features

- 🇬🇧🇧🇬 Dual-language display (English + Bulgarian)
- 📱 Mobile-first, touch-friendly design
- ♿ Accessible: keyboard navigation, ARIA labels, screen reader support
- 🎉 Celebration animations with confetti and star ratings
- ⭐ Score tracking: attempts and time
